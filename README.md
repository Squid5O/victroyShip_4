# FarmLife(팜라이프)
-------------------
## 기간
2024년 5월 02일 ~ 6월 25일

-------------------
## 개요
오픈월드 시뮬레이션 게임(농장 힐링)

-------------------
## 개발
언리얼 블루프린트(5.4v), 언리얼 C++(visual studio)
chatGPT4o, Python(server)

-------------------
## 기능
- 플레이어 인벤토리 시스템 구축
- chatGPT 를 활용한 NPC 페르소나 구현
- PCG 를 이용한 자동 레벨 디자인 구현
- 나이아가라 VFX 제작 및 활용
- 개별 농작물 성장 시스템 구현
- 플레이 시작 튜토리얼 제작
- 서버를 통한 AI 질문 대답 (SST,TTS) 적용
